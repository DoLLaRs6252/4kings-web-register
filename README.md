# # วิธีติดตั้ง

เลือก file path ที่ต้องการจะเก็บไฟล์ไว้
```
cd filepath
```

clone repo
``` 
git clone https://github.com/DoLLaRs6252/4kings-web-register.git
```

เปลี่ยน directory ไปที่ 4kings-web-register
```
cd 4kings-web-register
```

ติดตั้ง dependency
```
npm i
```
start server
```
npm run dev
```
